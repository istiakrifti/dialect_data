# Vashantor Dialect Viewer

A responsive web application for exploring Bengali dialect translations across 5 different regional dialects.

## Features

- **5 Bengali Dialects**: Barishal, Chittagong, Mymensingh, Noakhali, and Sylhet
- **10 Splits per Dialect**: Each containing 250 translation pairs
- **Paginated View**: Browse through 5 items at a time
- **Parallel Display**: View dialect text and standard Bengali side-by-side
- **Fully Responsive**: Works on mobile, tablet, and desktop devices

## How to Use

1. Select a dialect from the dropdown menu
2. Choose a split (1-10) to view 250 translation pairs
3. Navigate through pages using Previous/Next buttons
4. Each page shows 5 translation pairs at a time

## Local Development

```bash
# Clone the repository
git clone https://github.com/yourusername/vashantor-dialect-viewer.git

# Navigate to the directory
cd vashantor-dialect-viewer

# Start a local server
python -m http.server 8000

# Open browser to http://localhost:8000
```
